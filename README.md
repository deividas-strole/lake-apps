# Strole Digital Static Website

This is the simplest version of the website: no React, no Vite, no npm, no build step.

## Files

- `index.html` - the page structure
- `style.css` - all styling and animations
- `script.js` - page switching, form demo, and animation activation

## Run locally

Just double-click `index.html`, or use VS Code Live Server.

## Deploy to GitHub Pages

1. Create a GitHub repository.
2. Upload these files to the repository root.
3. Go to Settings → Pages.
4. Source: Deploy from branch.
5. Branch: `main` and folder `/root`.
6. Save.

Your site will be published by GitHub Pages.
